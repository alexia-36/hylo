# Hylo - HyperLocal

HyperLocal is a travel web application built with Next.js that helps users explore cities and countries around the world.

 Features
-  Current weather information
-  7-day weather forecast
-  Interactive maps
-  Image gallery for searched locations
-  Country and city information
-  Save favourite places
-  Mark places as visited
-  Rate visited destinations
-  User authentication with Supabase

 Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Prisma
- Supabase
- PostgreSQL
- OpenWeather API
- Open-Meteo API
- OpenStreetMap / Nominatim
- Unsplash API
